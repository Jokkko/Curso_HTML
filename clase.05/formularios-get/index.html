Formularios en html

Para crearlo se usa la etiqueta form

Sus atributos

  • action: Es la url a donde se va a enviar la informacion recopilada
  • metod: (protocolo http - metodos http) - Metodo POST y GET solo atraves de estas dos formas se puede enviar la info GET = a traves de la url POST = a traves de el cuerpo del html
  • target: Funciona igual que en los links (a /a) si pongo _blank en pesta nueva si pongo _self se abre en la misma pestania

Al enviarse el formulario crea una url (Query string)

Cuando envio la informacion a traves de GET el formulario arma la URL para enviar la informacion. A esa URL se la conoce como Query String

www.servidor(backend).net/?nombrevalor1=valor1&nombredevalor2=valor2
